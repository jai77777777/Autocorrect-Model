# Autocorrect-Model
A Python Flask web app that performs automatic spelling correction on text input and uploaded files using natural language processing.
Project Name: Autocorrect Model

Brief Description: 
The Autocorrect Model is a web app which we have developed to identify and correct spelling mistakes in the text put forward by the user or in uploaded text files. Our project is out to improve upon writing accuracy and at the same time to reduce the need for manual proofreading which is what we do by use of natural language processing techniques. 

What This Project:
Spills of wrong spelled words are typical in digital communication. This project we put forth a solution which is to present users with a easy to use platform to fix text inputs and documents which in turn is very useful for students, professionals and content creators. 

Features :
•	Spots and corrects spelling in as you type.
•	Supports the upload of text files for bulk correction. 
•	Simple and easy to use web interface. 
•	API support for integration of other applications. 

Tech Stack:
Programming language: Python. 
Backend Engine: Flask. Natural Language Processing 
Library: TextBlob. 
Front end: HTML, CSS, JavaScript. 

How to Get the Project Up and Running:.
1.	Install Python in your system. 
2.	Use this command to install the required libraries: pip install flask textblob. 
3.	Save the project file as app.py. 
4.	Run the app with: python app.py. 
5.	Open in your browser: http:/127.0.0.1:5000/
6.	Upload a.txt file or type out your text to see the corrected version.
